# 🤖 DocQnA — RAG-Based Document Q&A App

Ask natural language questions over your uploaded PDF documents using AI.

## Tech Stack
- **Frontend:** React 18 + TypeScript + Material UI
- **Backend:** ASP.NET Core 8 Web API
- **AI:** Microsoft Semantic Kernel + OpenAI GPT-4o
- **Vector DB:** Qdrant
- **Database:** PostgreSQL + EF Core
